crudFactory
service in module eightyServices

Description

A factory which creates a resource objects fot topic and question that lets you interact with RESTful server-side data sources.

Methods

  • allQuestions()

    Creates a resource object for requesting questions of current topic and it subtopics.

    Returns
    Object

    A resource "class" object with the following set of resource actions:

    • {object} query({Number} id, {Number} page, {Number} size, {string} sort) — Request for questions of topic with given id. page - page number, size - number questions per page, sort - question field for sorting

  • question()

    Creates a resource object for question.

    Returns
    Object

    A resource "class" object with the following set of resource actions:

    • {object} get({string=} id) — Request for question with given id.
    • {object} update({object} topic) — Request for update the given question.

  • topic()

    Creates a resource object for topic.

    Returns
    Object

    A resource "class" object with the following set of resource actions:

    • {object} get({string=} id) — Request for topic with given id.
    • {object} update({object} topic) — Request for update the given topic.
